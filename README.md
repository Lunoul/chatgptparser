# LLM parser

## Simple parser folders and files names for good LLM prompts


### How to run?

    $ git clone https://github.com/Lunoul/chatgptparser.git
    $ cd chatgptparser
    $ pip3 install tk
    $ python main.py

### How to compile to exe?

    $ pip install pyinstaller
    $ cd chatgptparser
    $ pyinstaller --onefile main.py
    $ cd dist
